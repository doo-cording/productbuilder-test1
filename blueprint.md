
# Rock Paper Scissors Game

## Overview

This project is a simple Rock Paper Scissors game implemented using HTML, CSS, and JavaScript.

## Design and Features

*   **Modern Design:** The game has a clean and modern design with a responsive layout.
*   **Interactive Elements:** The game has interactive buttons for the player to choose their move.
*   **Animations:** The game uses animations to provide feedback to the user.
*   **Custom Fonts:** The game uses custom fonts from Google Fonts.
